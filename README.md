# Exploring Iris Dataset 🌸

This project is a complete exploratory data analysis (EDA) on the famous Iris dataset using Python.
https://roadmap.sh/projects/exploring-iris-dataset
## Technologies Used
* Python
* Pandas
* Seaborn
* Matplotlib
* Jupyter Notebook

## Key Insights
* The dataset is perfectly balanced with 50 samples for each of the 3 species.
* The **Setosa** species is linearly separable from the other two.
* Petal length and petal width are highly positively correlated.

## How to run
1. Clone this repository.
2. Install requirements: `pip install pandas matplotlib seaborn jupyter`
3. Run `jupyter notebook` and open `iris_eda.ipynb`.
